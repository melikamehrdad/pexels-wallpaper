# wallpaper_project

A new Wallpaper Flutter project. This project can use on iOS, Android and web platform.

## Getting Started

This project is a starting point for a Flutter application.

## Features

* Show the wallpapers that fetch from the pexels API.
* Show an infinite grid view list.
* Select wallpaper and check details.
* Download wallpaper and save it to directory.
* Show a progress indicator, to check what percentage of the file is downloaded.
* Cache images.
